Welcome to the Proxy Server

Your tool for forwarding requests to any URL.

Usage

To use the proxy, make a request to /proxy/ with the following parameters:

  • url: The target URL to proxy the request to (required).
  • All other request parameters (method, headers, body) will be forwarded to the target URL.
